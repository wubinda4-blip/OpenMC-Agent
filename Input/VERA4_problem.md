# VERA Benchmark Problem 4: 三维 HZP 3×3 组件控制棒价值 (3D HZP 3x3 Assembly Control Rod Worth) — 问题描述

> **来源**: CASL-U-2012-0131-004《VERA Core Physics Benchmark Problems》Problem #4
> **反应堆**: WBN1 Cycle 1 初始堆芯中心 3×3 区域
> **状态**: 寿期初 (BOL) / 热态零功率 (HZP) 等温条件

---

## 1. 目的 (Purpose)

第四个 VERA 堆物理基准递进问题在三维组件问题基础上增加 **多个组件和 RCCA(棒簇控制组件)**。要求在不进行热工水力反馈和燃耗计算、且存在黑体中子吸收体的情况下预测 **特征值和棒功率分布**。本问题允许对实际堆芯一个区域的方法精度进行详细研究,并首次进行 **RCCA 移动和控制棒反应性价值计算**——这是常用于核方法验证的关键堆物理参数。

## 2. 问题构成 (Specifications)

问题由 **九个 Westinghouse 17×17 型燃料组件** 组成,按 **3×3 棋盘式** 直接取自 WBN1 初始装载图的中心。燃料处于寿期初 (BOL) 和热态零功率 (HZP) 等温条件。除标准材料外,本问题还测试在导向管中 **定义和放置 Pyrex、AIC、B₄C 吸收体** 的能力,以及通过给出组件的抽出步数来定位 RCCA 的能力。

## 12. 验证能力 (Capabilities)

成功完成本基准题应能演示:
- 离散可燃毒物簇的定义与放置
- 控制棒(RCCA)的定义、放置和自动轴向定位
- 多种组件类型的定义与布局
- 单一棒型中多种控制棒材料的定义
- 处理不位于网格边界的控制棒端头或材料边界
- 对毒物和控制棒等非燃料吸收体进行截面处理
- 考虑燃料、毒物、控制棒间轴向微小差异导致的"薄平面"效应
- 输出问题平均径向和轴向相对功率分布
- 计算微分与积分控制棒价值

## 13. 几何建模要求

- 9 个组件按 3×3 棋盘式布置,组件几何按上述规格尽可能显式建模。
- 含半显式定位格架、详细轴向反射层(气腔、端塞、间隙)。
- **径向所有边界:反射边界条件**(3×3 阵列外侧)。
- **轴向:真空(非重返)边界**,上下堆芯板外各加 **50 cm 慢化剂缓冲**。
- Pyrex 与控制棒按规格显式建模定位;套管塞放置于空导向管。
- 格架表示:格架质量均分到 289 个栅元,置于栅元外侧等效体积盒。
- 裂变率计数建议采用 ~49 层轴向网格(约 3 英寸区段,并显式分辨格架区)。
- 本问题宜按四分之一或八分之一对称运行。

## 14. 待求解量

对 12 个工况(基础 + 0–100% 棒抽出)分别计算:
1. **k-effective**(特征值);
2. **微分控制棒价值 (DRW)** 和 **积分控制棒价值 (IRW)**。

反应性价值公式:**ρ = (1/k₁ − 1/k₂) × 10⁵  [pcm]**

基础工况另需输出:
- 径向组件功率分布(9 个组件);
- 平均轴向功率分布。
