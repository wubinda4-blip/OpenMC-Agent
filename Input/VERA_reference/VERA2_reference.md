## 1. 参考解计算方法 (Reference Solution)

### 1.1 计算工具
参考解由 **SCALE 6.2 Beta KENO-VI**(CE 蒙特卡洛输运)计算。CSAS6 序列输入包括材料、密度、燃料同位素、精确几何描述等。对各燃料棒进行裂变率计数,归一化与后处理得到棒功率分布及不确定度分布。

### 1.2 截面
基于 **ENDF/B-VII.0 CE 截面**(SCALE 6.2 ce-v7-endf)。同时使用 565 K 和 600 K 截面。H-1 的 S(α,β) 数据仅 550 K、600 K 可用,565 K 工况作二次计算后手动插值。

### 1.3 计算参数
为降低功率分布不确定度,使用 **1.1×10⁹ 粒子**,1100 代、每代 1×10⁶ 粒子,跳过 100 代。特征值不确定度 < 3 pcm,最大功率分布不确定度 < 0.06%。

### 1.4 几何处理
栅元按同心圆柱显式建模;栅格按四分之一对称建模(含组件间隙);各可燃毒物与离散插入件按规格建模;所有边界反射。2Q 格架模型:每个栅元等量 Zircaloy 质量(总/289)置于外圈等效体积盒。结果按八分之一对称折叠(对称棒功率取平均,对称 σ 取平均后除以 √2)。

### 1.5 计算平台
SCALE 6.2 Beta 2 在 INL Fission 超算上执行,300 核约 3 小时,每核内存达 2.7 GB。
