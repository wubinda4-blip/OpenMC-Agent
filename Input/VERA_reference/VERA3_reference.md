## 1. 参考解计算方法 (Reference Solution)

### 1.1 计算工具
参考解由 **SCALE 6.2 Beta KENO-VI**(CE 蒙特卡洛输运)计算。CSAS6 序列输入包括材料、密度、燃料同位素、精确几何描述等。对各燃料棒在规定轴向位置进行裂变率计数,归一化后处理得到裂变率分布与不确定度。仅在 600 K 和 565 K 可用。

由于问题规模与细节(含半显式格架、每个功率区需独立单元),编写了 FORTRAN 程序基于简单问题描述自动生成输入,代码位于 `/home/agm/git/kenogen`。

### 1.2 截面
基于 **ENDF/B-VII.0 CE 截面**(SCALE 6.2 ce-v7-endf)。同时使用 565 K 和 600 K 截面。H-1 的 S(α,β) 仅 550 K、600 K 可用,565 K 工况作二次计算后手动插值。

### 1.3 材料
燃料同位素基于同位素方程计算(Table P3-2);间隙显式建模为氦;硼浓度按重量份额输入(1300 ppm → 0.0013/0.9987;1066 ppm → 0.001066);上下 nozzle 与堆芯板按密度和高度手动均匀化。

### 1.4 计算参数
使用 **25×10⁹ 粒子**,5000 代、每代 5×10⁶ 粒子,跳过 500 代。特征值不确定度 < 0.6 pcm,最大功率不确定度 < 0.16%,所有位置平均棒功率不确定度 < 0.04%。

### 1.5 几何处理
组件几何按规格尽可能显式建模;径向反射边界;上下堆芯板外加 50 cm 慢化剂;格架表示同 Problem 2Q;裂变率计数在 49 层轴向网格(约 3 in 区段并显式分辨格架区)。

### 1.6 计算平台
输入超过 54,000 行。SCALE 6.2 Beta 2 在 INL Fission 超算上执行,240 核运行约 5 天,每核内存达 4 GB。
