## 1. 参考解计算方法 (Reference Solution)

### 1.1 计算工具
参考解由 **SCALE 6.2 Beta KENO-VI**(CE 蒙特卡洛输运)计算。CSAS6 序列输入包括材料、密度、燃料同位素、精确几何描述等。对各燃料棒在规定轴向位置进行裂变率计数。

由于问题规模与细节,编写 FORTRAN 程序自动生成输入(~450,000 行),代码位于 `/home/agm/git/kenogen`。

### 1.2 截面
基于 **ENDF/B-VII.0 CE 截面**(SCALE 6.2 ce-v7-endf),仅使用 565 K 截面。H-1 的 S(α,β) 仅 550 K、600 K 可用,作二次计算并应用 -40 pcm 手动校正因子。

### 1.3 材料
燃料同位素基于同位素方程计算(Table P4-2);间隙显式为氦;硼浓度按重量份额(1360 ppm → 硼 0.001360,水 0.998640);上下 nozzle 与堆芯板手动均匀化。

### 1.4 计算参数
- **基础工况**(RCCA 在 257.9 cm):50×10⁹ 粒子,10,000 代、每代 5×10⁶ 粒子,跳过 500 代。特征值不确定度 < 0.4 pcm,平均棒功率不确定度 < 0.065%。
- **棒价值工况**:1/10 粒子数,即 5×10⁹ 粒子(1000 代,跳过 300)。特征值不确定度 < 1.5 pcm,平均棒功率不确定度 < 0.25%(最大 < 3.9%)。

### 1.5 几何处理
Pyrex 与控制棒显式建模定位;套管塞放置于空导向管;格架表示同 Problem 2Q/3;裂变率计数在 49 层轴向网格。所有工况特征值应用 -40 pcm 的 H-1 在 550 K 散射数据校正。

### 1.6 计算平台
输入近 450,000 行。SCALE 6.2 Beta 2 在 INL Fission 超算执行。基础工况 300 核约 11 天(每核 4.3 GB);棒价值工况各约 26 小时(300 核,4.3 GB/核)。
