## 1. 目的 (Purpose)

第二个 VERA 堆物理基准题演示对 **二维燃料棒阵列(燃料栅格)** 建模的能力,该栅格是 PWR 燃料组件轴向中心区的典型结构。除增殖因子外,结果还要求比较各燃料棒之间归一化的裂变反应率分布(以下简称"棒功率" pin powers)。

## 2. 问题构成 (Specifications)

问题由 **单个 Westinghouse 17×17 型燃料栅格** 构成,处于寿期初 (BOL)。燃料本身参数见第 6 节(燃料棒几何)与第 7.1 节(栅格)。材料包括:**UO₂、Zircaloy-4、水(含可溶硼)**;导向管内可插入的中子毒物:**银-铟-镉 (AIC)、碳化硼 (B₄C)、Pyrex(硼硅玻璃 B₂O₃-SiO₂)、B₄C-Al₂O₃(WABA)**;不锈钢 304 用于仪表管套管;整体可燃毒物 **IFBA** 和 **钆(Gadolinia)**。

本题分为 **17 个计算工况**:
- **2A**:典型的零功率等温条件;
- **2B、2C、2D**:相同几何但燃料温度取满功率运行典型范围(与 Problem 1 一致);
- **2E ~ 2P**:检验对不同可燃毒物和控制棒类型产生的径向不均匀性的建模能力;
- **2Q**:检验对带均匀分布质量的定位格架产生的反应性下降和径向功率分布的建模能力。

## 6. 燃料棒几何与统一建模约定 (Table 1, §1.1)

### 6.1 径向几何

普通燃料棒不是“燃料直接接触包壳”，而是由燃料芯块、氦气间隙、Zircaloy-4 包壳和外部慢化剂组成。所有半径均从棒中心轴量起。

| 径向范围 | 厚度 | 填充材料 | 建模含义 |
|---|---:|---|---|
| `0 ≤ r < 0.4096 cm` | — | UO₂ 燃料 | 3.1%、3.6% 或 1.8% 富集燃料，按工况选择 |
| `0.4096 ≤ r < 0.418 cm` | 0.0084 cm = 0.084 mm | 氦气 | 芯块—包壳间隙；不可误填为水 |
| `0.418 ≤ r < 0.475 cm` | 0.057 cm = 0.57 mm | Zircaloy-4 | 燃料棒包壳 |
| `r ≥ 0.475 cm` 至栅元边界 | — | 含硼慢化剂 | 单棒方形栅元半宽为 `1.26/2 = 0.63 cm` |

> **OpenMC/IR 约束**：燃料芯块、氦隙和包壳必须是三个不重叠的同心区域。不能把 `0.4096–0.475 cm` 整体视为包壳，也不能省略氦隙后把芯块半径扩展到包壳内表面。

## 7. 管件、可燃毒物与控制棒的详细几何

### 7.0 通用嵌套关系：插入件不是导向管本身

24 个导向管位置中的任意结构都由“插入件 + 插入件外水隙 + Zircaloy-4 导向管壁 + 导向管外慢化剂”共同组成。插入 Pyrex、WABA 或控制棒时，**不得删除导向管壁**。

```text
棒中心
  插入件本体                    0 ≤ r < 0.484 cm（具体内部分层见后文）
  含硼慢化剂水隙              0.484 ≤ r < 0.561 cm
  Zircaloy-4 导向管壁          0.561 ≤ r < 0.602 cm
  含硼慢化剂                  r ≥ 0.602 cm，直到方形栅元边界
```

插入件外半径为 0.484 cm，导向管内半径为 0.561 cm，因此二者之间始终保留：

`0.561 - 0.484 = 0.077 cm = 0.77 mm`

的含硼水隙。该水隙对慢化和局部功率分布有直接影响，不能与 SS304/Zircaloy-4 均匀化后省略。

- Problem 2 的二维 2E/2F/2J/2K 只采用有效毒物段横截面；

- 若端塞与组件顶部结构因原始估算尺寸出现毫米级不闭合，应显式记录几何闭合假设，不要偷偷拉伸毒物高度。

### 7.4 RCCA 控制棒 (Table 8, §1.6)

WBN1 Cycle 1 的真实 RCCA 是**轴向混合控制棒**：下端为 AIC，较长的上部为 B₄C。AIC 和 B₄C 不是径向套在一起，而是在同一根 SS304 包壳内沿轴向首尾相接。

#### 7.4.1 AIC 截面的径向分层

| 径向范围 | 填充材料 |
|---|---|
| `0 ≤ r < 0.382 cm` | AIC：80/15/5 wt% Ag/In/Cd，密度 10.2 g/cc |
| `0.382 ≤ r < 0.386 cm` | 氦气间隙 |
| `0.386 ≤ r < 0.484 cm` | SS304 控制棒包壳 |
| `0.484 ≤ r < 0.561 cm` | 含硼慢化剂水隙 |
| `0.561 ≤ r < 0.602 cm` | Zircaloy-4 导向管壁 |
| `r ≥ 0.602 cm` | 含硼慢化剂 |

#### 7.4.2 B₄C 截面的径向分层

| 径向范围 | 填充材料 |
|---|---|
| `0 ≤ r < 0.373 cm` | 100% B₄C，密度 1.76 g/cc |
| `0.373 ≤ r < 0.386 cm` | 氦气间隙 |
| `0.386 ≤ r < 0.484 cm` | SS304 控制棒包壳 |
| `0.484 ≤ r < 0.561 cm` | 含硼慢化剂水隙 |
| `0.561 ≤ r < 0.602 cm` | Zircaloy-4 导向管壁 |
| `r ≥ 0.602 cm` | 含硼慢化剂 |

两段使用相同的包壳内、外半径，因此轴向接合处的棒件外形连续；B₄C 芯体半径较小，只是其氦气间隙更厚。

#### 7.4.3 完整 RCCA 的轴向分段

完全插入时，毒物下端位于 `z = 17.031 cm`：

| 轴向段 | 高度 | 完全插入时的范围 | 材料 |
|---|---:|---:|---|
| 下部端头 | 101.6 cm | `17.031–118.631 cm` | AIC |
| 上部吸收段 | 259.08 cm | `118.631–377.711 cm` | B₄C |
| 上部气腔 | 10.7 cm | 名义上 `377.711–388.411 cm` | 氦气 |
| 端塞 | 约 1.9 cm | 端部简化 | SS304 |

控制棒每退出一步上移 1.5875 cm，最多 230 步。若 `N` 为退出步数，则毒物下端位置为：

`z_bottom(N) = 17.031 + 1.5875 × N  cm`

AIC/B₄C 分界面和毒物上端随整根棒同时平移，不能只移动 B₄C 段。

#### 7.4.4 Problem 2 的 2G/2H 与真实混合 RCCA 的区别

Problem 2 是二维中央燃料区截面，没有轴向坐标。其两个工况用于分别检验吸收材料的径向截面：

- **2G：24 AIC**——24 个导向管位置均使用 §7.4.1 的 AIC 截面；
- **2H：24 B₄C**——24 个导向管位置均使用 §7.4.2 的 B₄C 截面。

因此，生成 Problem 2 的二维模型时，不应强行把 2H 改造成“AIC 下段 + B₄C 上段”的三维 RCCA；生成 Problem 3–5 等三维模型时，则必须恢复 §7.4.3 的轴向混合结构。

导向管塞用于减少没有 Pyrex 或 RCCA 插入件的空导向管旁流，通常位于活性燃料区上方，因此 Problem 2 的二维中央截面不建模。

仪表套管插入中心仪表管 `(9,9)`，自身是厚壁 SS304 空心管。它不放在 24 个导向管位置，也不与 Pyrex/RCCA 共用位置。

### 7.7 IFBA 整体可燃毒物燃料棒 (Table 11–12, §1.9)

IFBA 不是导向管插入件，而是直接涂覆在 UO₂ 芯块外表面的极薄 ZrB₂ 层。涂层不取代 UO₂，只占用原燃料—包壳氦隙的一小部分。

#### 7.7.1 IFBA 有涂层段的径向分层

10 μm = 0.001 cm，因此涂层外半径为：

`0.4096 + 0.0010 = 0.4106 cm`

| 径向范围 | 填充材料 |
|---|---|
| `0 ≤ r < 0.4096 cm` | UO₂ 燃料 |
| `0.4096 ≤ r < 0.4106 cm` | ZrB₂ IFBA 涂层 |
| `0.4106 ≤ r < 0.418 cm` | 氦气间隙 |
| `0.418 ≤ r < 0.475 cm` | Zircaloy-4 包壳 |
| `r ≥ 0.475 cm` | 含硼慢化剂 |

| IFBA 材料参数 | 值 |
|---|---:|
| B-10 线性装载量 | 2.355 mg/in |
| B-10 富集度 | 50% |
| 涂层密度 | 3.85 g/cc |
| B-10 | 9.347 wt% |
| B-11 | 9.347 wt% |
| Zr | 81.306 wt% |

薄层几何必须使用 `r = 0.4096 cm` 与 `r = 0.4106 cm` 两个独立圆柱面；不能把 IFBA 与燃料均匀混合，也不能为了网格方便把 10 μm 涂层任意加厚而不保持 B-10 线性装载量。

#### 7.7.2 IFBA 轴向范围

涂层高度 304.8 cm，并在 365.76 cm 燃料柱中轴向居中。上下未涂覆燃料各长：

`(365.76 - 304.8) / 2 = 30.48 cm`

结合燃料柱下端 `z = 11.951 cm`，IFBA 涂层范围为：

- 下端：`11.951 + 30.48 = 42.431 cm`
- 上端：`42.431 + 304.8 = 347.231 cm`

完整三维 IFBA 棒应分成“下部普通燃料段—中央 IFBA 涂层段—上部普通燃料段”。Problem 2L/2M/2N 表示中央轴向截面，所选 IFBA 燃料棒使用有涂层截面。

### 7.11 几何与材料校验清单

1. 所有半径必须严格递增，任意相邻层之间不得重叠或留下未填充区域。
2. Pyrex/WABA/AIC/B₄C 插入件外半径均为 0.484 cm；其外仍必须存在 0.077 cm 慢化剂水隙和 Zircaloy-4 导向管壁。
3. Pyrex 的吸收材料只在 `0.241–0.427 cm`；WABA 的吸收材料只在 `0.353–0.404 cm`。
4. WABA 中心 `r < 0.286 cm` 是慢化剂；Pyrex 中心 `r < 0.214 cm` 是氦气；二者不可混用。
5. IFBA 是 10 μm 的 ZrB₂ 表面涂层；Gadolinia 是 UO₂ 内的均匀混合物。
6. 仪表套管中心是真空，套管外与原仪表管之间仍有慢化剂水隙。
7. Problem 2 的 2G/2H 是二维 AIC/B₄C 截面工况；真实三维 RCCA 才使用下 AIC、上 B₄C 的轴向拼接。
8. 结构材料必须使用 SS304、Zircaloy-4 等合金材料卡，不得无标记地替换为纯元素。
9. 2D 模型只取中央燃料区横截面；端塞、气腔、控制棒移动和 IFBA 上下未涂覆段属于 3D 模型语义。
10. 任一插入件 universe 的最外部区域必须填充到方形栅元边界，避免 OpenMC 出现未定义空间或重叠 cell。
