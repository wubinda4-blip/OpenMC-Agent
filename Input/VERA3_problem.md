# VERA Benchmark Problem 3: 三维 HZP 燃料组件 (3D HZP Assembly) — 问题描述

> **来源**: CASL-U-2012-0131-004《VERA Core Physics Benchmark Problems》Problem #3
> **反应堆**: WBN1 Cycle 1 初始堆芯
> **状态**: 寿期初 (BOL) / 热态零功率 (HZP) 等温条件

---

## 1. 目的 (Purpose)

本基准题演示对 **三维燃料组件**(PWR 反应堆分析中的典型结构)的计算能力。要求在不进行热工水力反馈和燃耗计算的情况下预测 **特征值和棒功率分布**。

本题重点是演示对 **定位格架对中子通量影响** 的分辨能力,并开始建模燃料栈上下的非燃料结构材料及相应边界条件。这是第一个三维问题,需要在非重返(真空)边界条件下定义 **轴向中子反射层区**。径向上按四分之一对称处理。

## 2. 问题构成 (Specifications)

问题由 **单个 Westinghouse 17×17 型燃料组件** 构成,处于寿期初 (BOL) 和热态零功率 (HZP) 等温条件。材料为:**UO₂ 燃料、Zircaloy-4 包壳、Inconel-718(端部格架)、Stainless Steel 304 和水(含可溶硼)**。

本题分为 **两个计算工况**(差异见第 3 节)。

## 15. 验证能力 (Capabilities)

成功完成本基准题应能演示:
- 支持多个轴向燃料区
- 支持显式(气腔)和均匀化(nozzle)轴向反射层与真空边界
- 对多种定位格架类型进行轴向布置和材料均匀化
- 考虑定位格架对截面处理的影响
- 输出组件级功率分布(1D/2D/3D 功率、轴向偏移)
- 输出峰值相对棒功率统计(FdH、Fq)及位置

## 16. 几何建模要求

- 组件几何按上述规格尽可能显式建模,包括半显式定位格架、详细轴向反射层(气腔、端塞、间隙)。
- **径向所有边界:反射边界条件**。
- **轴向:真空(非重返)边界**,上下堆芯板外各加 **50 cm 慢化剂缓冲**,以保证轴向泄漏精确计算。
- 定位格架:将格架质量均分到 289 个栅元,在栅元外侧等效体积盒放置,忽略格架套筒。
- 3B 的 Pyrex 按 16 根布置(参考 §1.5 的环状配置);不含 Pyrex 的导向管放置套管塞。
- 裂变率计数建议采用 ~49 层轴向网格(约 3 英寸区段,并显式分辨格架区)。

## 17. 待求解量

对 3A 和 3B 两个工况分别计算:
1. **k-effective**(特征值);
2. **轴向偏移 (Axial Offset)**;
3. **径向棒功率分布**(归一化,平均 1.0);
4. **轴向功率分布**(沿 49 层网格)。
