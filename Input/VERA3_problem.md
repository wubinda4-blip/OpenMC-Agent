## 1. 目的 (Purpose)

本基准题用于建立单个 Westinghouse 17×17 型 PWR 燃料组件的三维 HZP 模型，并在无热工水力反馈、无燃耗条件下计算特征值和功率分布。

本题显式包含燃料栈上下非燃料结构、轴向反射层、定位格架和 3B 工况的 Pyrex 可燃毒物棒及套管塞。径向边界采用反射边界；轴向边界采用真空边界。

---

## 2. 问题构成 (Specifications)

问题由 **单个 Westinghouse 17×17 型燃料组件** 构成。材料包括 UO₂、Zircaloy-4、Inconel-718、SS304、氦气和含硼水。

本题包含两个工况：

| 工况 | 描述 |
|---|---|
| 3A | 无 Pyrex、无套管塞 |
| 3B | 16 根 Pyrex 可燃毒物棒；其余未被 Pyrex 占据的 8 个导向管位置放置套管塞 |

### 5.5 `F/G/I/P/T` 的三维语义（重要）

上述 pin map 只定义 **289 条轴向栅元路径的类别与坐标**，不是一个可沿全高直接拉伸的单一二维 universe：

- `F`：该位置是一条燃料棒路径；其内部材料随 `z` 在下端塞、燃料、上端塞、氦气腔之间变化。
- `G`：该位置是一条普通导向管路径；导向管壁连续存在，内部通常为含硼水。
- `P`：该位置是一条被选作 Pyrex 的导向管路径；只有 Pyrex 棒实际占据的轴向区间使用 Pyrex 截面，其余高度仍是普通水填充导向管。
- `T`：该位置是一条被选作套管塞的导向管路径；只有 `z=383.31–394.31 cm` 存在 SS304 套管塞，其余高度仍是普通水填充导向管。
- `I`：中心仪表管路径；Problem 3 不含 §1.8 所述的 SS304 仪表套管，因此内部按含硼水处理。

因此，不得把 3B 的 16 个 `P` 和 8 个 `T` universe 从下 nozzle 顶面一直拉伸到上 nozzle 底面。三维装配必须在每个相关轴向切面重新选择对应的栅元截面。

### 6.2 三维对象在轴向上的连续性

必须区分“基础轴向层”和“穿过基础层的棒/管路径”：

1. **燃料棒路径**从 `z=10.281` 开始，到 `z=395.381` 结束；在这一路径内依次为下端塞、燃料段、上端塞和气腔。
2. **24 根导向管路径**在详细组件区 `z=6.053–397.510` 内保持 `Zr-4 管壁 + 内部流道`，不能因燃料棒轴向材料改变而消失。
3. **中心仪表管路径**在详细组件区保持水填充的 Zr-4 管截面；本题不插入 SS304 仪表套管。
4. **定位格架**是局部轴向覆盖层，只占各栅元最外侧的等效方框区域；其余燃料、包壳、管壁和插入件继续贯穿。
5. **Pyrex 与套管塞**只改变被选中导向管的内部区域，绝不能替换导向管的 Zircaloy-4 管壁。
6. **nozzle/堆芯板**是整平面均匀化 slab。进入这些 slab 后，不再同时放置详细 17×17 lattice，否则会双重计数结构材料。

### 9.2 质量守恒的等效几何

每个格架的质量平均分配给 289 个栅元。对每个栅元，在格架高度内需要的格架材料横截面积为：

```text
A_cell = grid_mass / (material_density × 289 × grid_height)
```

由本题密度得到：

| 格架 | 每栅元质量 | 每栅元材料横截面积 `A_cell` | 若采用对称方形边框，等效边框厚度 |
|---|---:|---:|---:|
| 端部 Inconel-718 | 3.519 g | 0.111142 cm² | 0.02245 cm（0.2245 mm） |
| 中间 Zr-4 | 3.028 g | 0.121138 cm² | 0.02451 cm（0.2451 mm） |

对称方形边框的计算方式为：在 `1.26×1.26 cm²` 栅元最外侧放置材料，内方孔边长满足 `1.26²-a_inner²=A_cell`。也可以使用其它“位于栅元外侧”的等体积小方框，但必须满足相同材料体积。

### 9.3 叠加规则

- 格架材料只能放入每个 pitch cell 最外侧原本为慢化剂的区域。
- 燃料、包壳、导向管、仪表管、Pyrex 和套管塞均继续贯穿格架层。
- 不得把整个格架高度层均匀化为 17×17 全区域的 Zr-4/Inconel，也不得用格架材料替换燃料或管内材料。
- 忽略 spacer sleeve；不要在 24 个导向管和中心仪表管周围另加套筒质量。
- 下端部格架的下边界被有意移到燃料栈底面 `z=11.951 cm`。

nozzle 的体积分数使用整个 `21.50×21.50 cm²` 组件横截面计算：

```text
f_SS = mass_SS / (rho_SS304 × assembly_pitch² × nozzle_height)
f_coolant = 1 - f_SS
```

采用 `rho_SS304=8.00 g/cc`。对应的简单体积加权混合密度约为：下 nozzle `2.7693 g/cc`，上 nozzle `2.1325 g/cc`，堆芯板 `4.3715 g/cc`。在 OpenMC 中更推荐按体积分数混合两种成分，而不是只设置一个无组成意义的总密度。

### 11.2 横截面分布语义

这些构件在本基准中不是详细 CAD 几何：

- 在各自 z 范围内，**整个组件横截面**填充对应均匀混合物。

## 17. 几何建模要求与自动校验

### 17.1 推荐的装配顺序

1. 建立本工况材料：燃料、含硼水、氦、Zr-4、SS304、Inconel-718、Pyrex。
2. 建立不同轴向状态的基础 pin universe：燃料段、下/上端塞、气腔、空水栅元、普通导向管、中心仪表管。
3. 对 3B 建立 Pyrex 毒物段、Pyrex 气腔段和套管塞段的导向管 universe。
4. 按 §6 的所有切面形成轴向分段，每段选择正确的 17×17 lattice。
5. 在 8 个格架高度段叠加外圈等效方框，不改变贯穿棒/管内部。
6. 在 `z<6.053 cm` 和 `z≥397.510 cm` 切换为整平面均匀化 slab/慢化剂 slab。
7. 最后设置径向反射、轴向真空边界及 tally 网格。

### 17.2 必须满足的几何不变量

- 每个详细 lattice 横截面仍有 `17×17=289` 个路径；264 条燃料棒路径、24 条导向管路径、1 条仪表管路径。
- 3B 的 16 个 `P` 与 8 个 `T` 坐标互不重叠，且并集恰为全部 24 个导向管坐标。
- 插入件不能删除或覆盖 `0.561–0.602 cm` 的 Zr-4 导向管壁。
- Pyrex 毒物必须是 `0.241–0.427 cm` 的环形层，而不是实心 Pyrex 圆柱。
- 套管塞必须是 `383.31–394.31 cm` 的短 SS304 实心圆柱，不得延伸进活性燃料区。
- 格架层中燃料、包壳、管壁和插入件数量与非格架层相同；仅慢化剂外圈的一部分被格架材料替代。
- nozzle/堆芯板层不得继续引用 17×17 lattice。
- Problem 3 不含控制棒、IFBA、WABA、钆棒或 SS304 仪表套管。
- 上下 50 cm 水缓冲层外表面是真空；水缓冲层本身不是真空。

### 17.3 常见错误及阻断条件

| 错误 | 结果 |
|---|---|
| 把 `P/T` pin map 沿全高拉伸 | 在活性区错误加入套管塞，或在肩部/nozzle 重复加入 Pyrex |
| 用整层均匀化格架材料替代 lattice | 切断燃料棒和导向管的 through-path |
| 插入 Pyrex 后删除导向管壁 | 低估 Zr-4 并扩大水/毒物体积 |
| 把 Pyrex 建成实心棒 | 丢失中心腔、内管和环形毒物自屏蔽效应 |
| 在 nozzle 中同时保留详细 lattice | 结构材料与冷却剂双重计数 |
| 把中心仪表管填真空/SS 套管 | 引入 Problem 3 未指定的结构 |
| 将结构合金近似为纯元素且未标记 | 偏离参考材料吸收与散射特性 |

## 17.4 面向 IR/Agent 的对象分类

建议把本题表达为以下对象，而不是一个巨大的逐层完整 lattice：

| 对象类别 | 内容 |
|---|---|
| `through_paths` | 264 根燃料棒、24 根导向管、1 根仪表管 |
| `localized_inserts` | 3B 的 Pyrex 毒物/气腔段、8 个短套管塞 |
| `axial_overlays` | 8 个定位格架外圈等效方框 |
| `homogenized_slabs` | 上下 nozzle、上下堆芯板 |
| `moderator_slabs` | 上下肩间隙、堆芯板外 50 cm 缓冲层 |

渲染器应先保持 through-path，再按 z 区间加入 localized insert 和 overlay；不能让 overlay 或 slab 隐式吞掉贯穿结构。
